# Otome Game Research Workspace

This repository organizes the research materials produced in this Codex chat.

Canonical research files are kept under `research/`. The original generated
files in `outputs/` and intermediate workspace files in `work/` are preserved
as-is and treated as working copies.

## Directory Map

- `research/literature/`: literature matrices, workbook previews, and source/audit data.
- `research/methods_review/`: methods-review issue table and next-step notes.
- `research/writing_scaffold/`: claim ledger, writing scaffold, and AI-use ledger.
- `research/study_design/cluster_c_survey_route/`: earlier Cluster C survey-route design materials.
- `research/study_design/cluster_c_text_analysis_route/`: current Cluster C text-analysis route using scraped comments/reviews.
- `research/manuscript_scaffold/`: Word and LaTeX manuscript scaffold files.
- `research/scripts/`: scripts used to generate workbooks, fetch abstracts, and build manuscript scaffolds.

## Current Research Focus

Topic: otome games, emotional value, emotional consumption, digital intimacy,
and business/marketing research.

Current preferred route: Cluster C, using scraped comments/reviews and text
analysis rather than questionnaire survey data.
