# Scraping And Public Text Ethics Note

This project plans to analyze public comments/reviews about otome games.

Before collecting any platform data:

- Use only public content that can be accessed without bypassing login walls, CAPTCHA, paywalls, rate limits, or anti-scraping systems.
- Prefer official APIs, official exports, or manually downloaded public exports when available.
- Do not collect private messages, private groups, non-public accounts, or personally sensitive data.
- Hash or remove user identifiers before analysis.
- Avoid publishing long verbatim comments that can be searched back to individual users.
- Store source URLs in internal audit files only when needed for verification.
- Route low-confidence or potentially identifying examples to manual review.

The current repository does not yet contain raw public comment/review data.
