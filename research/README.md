# Research Folder Guide

This folder is the clean, Git-managed research archive for the otome game
project.

## 1. Literature

`literature/matrices/` contains the literature matrix workbook. The current
version keeps original English abstracts and Chinese translations.

`literature/previews/` stores workbook preview images.

`literature/audit/` stores source/audit material used while collecting article
metadata.

## 2. Methods Review

`methods_review/` contains the methods-review issue table and next-step plan.
Use this folder when checking identification, data feasibility, construct
validity, overclaiming risks, and reporting gaps.

## 3. Writing Scaffold

`writing_scaffold/` contains claim ledgers and structured writing scaffolds.
These files are meant to support author writing rather than replace it.

## 4. Study Design

`study_design/cluster_c_survey_route/` keeps the earlier survey/mixed-methods
route for comparison.

`study_design/cluster_c_text_analysis_route/` is the current route: scraped
comments/reviews plus text analysis, aligned with business/marketing questions
about emotional value, emotional consumption, and digital intimacy.

## 5. Manuscript Scaffold

`manuscript_scaffold/` contains the Word and LaTeX manuscript skeletons.

## 6. Scripts

`scripts/` contains reproducibility scripts used to build matrices and document
scaffolds. Treat these as provenance helpers; polish them before using them as
formal replication code.
